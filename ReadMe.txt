Project Title: Idol Voting System
Module: Admin
Description: The admin is responsible for creating new votees (singing idols) and starting a new poll. The admin needs to create, read, update, and delete (CRUD) votees, as well as start and end polls. The admin also needs to be able to view the list of registered voters and the current poll status.


*Steps to run the project*
1. Run the 'AdminMenu.java'.
2. Select a choice to perform an action (login, votees, polls).
3. Type '0' to exit the program.
